2.6 OFDM/OFDMA Downlink

OFDMA is how the base station shares the downlink grid between phones

OFDM gives us a grid of subcarrier-like tiles. OFDMA turns that grid into a shared downlink: each phone hears the same wide signal, but the scheduler assigns different data tiles to different phones according to channel quality.

Simplified downlink grid

This demo uses a simplified 64-tile OFDM grid. Only 48 tiles carry user data; the remaining tiles represent pilot/reference, guard, and control overhead.

Before a phone decodes the data grid, it receives a scheduling grant: a short control instruction that says which assigned data tiles to read and which modulation to expect.

Drag the phones. Distance from the base station lowers channel quality. The gray half adds a shadowing penalty, so two phones at the same radius can still receive different allocations.
